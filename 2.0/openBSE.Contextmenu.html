Class: Contextmenu

openBSE. Contextmenu

上下文菜单类

new Contextmenu(bulletScreenEngine, element [, layer] [, pause])

上下文菜单对象。用于实现一个弹幕上下文菜单。
Parameters:
Name Type Argument Default Description
bulletScreenEngine openBSE.BulletScreenEngine 弹幕引擎对象：一个弹幕 openBSE.BulletScreenEngine 对象。要添加上下文菜单的
element Element 上下文菜单元素：当显示上下文菜单时要显示的 div 。有关 Element 接口的信息请参阅MDN Element 。
layer number <optional>
 10 弹幕层级：当显示上下文菜单或鼠标指向弹幕时弹幕要移动到的层级。有关弹幕层级的详细说明请参阅 openBSE~options 结构。
pause boolean <optional>
 true 是否暂停：当鼠标指向弹幕或单开上下文菜单时弹幕是否暂停移动/播放。
