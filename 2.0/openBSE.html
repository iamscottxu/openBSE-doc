BulletScreen

BulletScreen 结构用于存放单条弹幕数据。
Type:
  • object
Properties:
Name Type Argument Default Description
text string 弹幕文本
canDiscard boolean <optional>
 true 是否允许丢弃：（此参数在事件中修改无效）在弹幕过多时，程序将自动丢弃一些延迟过高的弹幕。此选项为 false 时本条弹幕无论如何都不会被丢弃，使用本选项的场景如本用户发送的弹幕。（注意：不要将太多的弹幕的 canDiscard 设为 false， 否则会因超时的弹幕不会被丢弃而造成意外的问题。）
startTime number <optional>
 options.clock() 弹幕进入时间：（此参数在事件中修改无效）单位：毫秒，默认为时间基准（options.clock）当前时间。
type openBSE.BulletScreenType <optional>
 openBSE.BulletScreenType.rightToLeft 弹幕类型：（此参数在事件中修改无效）一个类型为 openBSE.BulletScreenType 的枚举。
style openBSE~BulletScreenStyle 弹幕样式：一个 openBSE~BulletScreenStyle 结构。设置此选项中的任何一个值，将覆盖对应的全局设置。
layer number <optional>
 0 弹幕层级：此参数越大，弹幕越靠前。一条弹幕在比它层级小的弹幕前面，在比它层级大的弹幕后面。如果层级相同按照进入时间确定层级顺序。
more... any 其他自定义字段：（在事件中修改修改此参数无需将 e.redraw 设置为 true）例如 uuid 、 id 等。（注意：因为在事件响应方法中返回的弹幕对象是原对象克隆的，所以无法直接比较，必须使用自定义字段唯一标识一条弹幕。）
